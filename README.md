## 设计风格

- **背景**: 深色网格底 (#0A0A0F) + 径向渐变
- **强调色**: 荧光绿 (#00FF88)
- **警示色**: 红色 (#FF3366)
- **金色**: 数据/金句 (#FFD700)
- **字体**: Noto Sans SC (中文)
- **底部**: 预留 20% 主播位

## 使用说明

1. **最终视频**: `renders/video.mp4` (渲染完成后)
2. **添加主播**: 在底部 20% 区域叠加主播圆形头像
3. **字幕**: 已按口播稿分段，可在剪辑软件中添加字幕
4. **修改**: 编辑 `index.html` 中的文本内容即可调整画面

## 管线流程

```
用户文本 → 编导拆解 → 分镜设计 → HyperFrames渲染 → 最终MP4
     ↑          ↑           ↑            ↑
  video-   storyboard-   visual-    video-editor
  director   artist      analyst    (后期精修)
```

## 备注

- 渲染使用 HyperFrames v0.7.21
- 由于沙箱内存限制 (8GB)，使用 standard 质量渲染
- 如需更高质量，可在本地运行 `hyperframes render --quality high`
